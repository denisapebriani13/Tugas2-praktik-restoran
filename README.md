# Tugas 2 Praktik - Aplikasi Restoran

## Deskripsi
Program Java berbasis console untuk aplikasi restoran sederhana. Program ini mendukung pemesanan makanan/minuman oleh pelanggan serta fitur pengelolaan menu oleh pemilik restoran.

## Fitur
- Input dan tampilan menu menggunakan **Array**
- Proses pemesanan pelanggan (bisa pesan banyak item)
- Perhitungan otomatis: subtotal, pajak 10%, biaya layanan Rp20.000
- Diskon 10% jika subtotal > Rp100.000
- Promo "Beli 1 Gratis 1" untuk minuman jika subtotal > Rp50.000
- Kelola Menu (Tambah, Ubah, Hapus)
- Validasi input

## Teknologi
- Java
- OOP (Class & Object)
- Array
- Scanner untuk input

## Cara Menjalankan
1. Buka project di **Apache NetBeans**
2. Jalankan file `Main.java`
3. Pilih menu sesuai kebutuhan

## Struktur File
- `Menu.java` → Class untuk objek menu
- `Main.java` → Kelas utama program
